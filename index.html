Tomasulo's Algorithm for Out of Order Execution

This project simulates a fully pipelined CPU that implements Tomasulo’s Algorithm for out-of-order execution of instructions. The objective was to create a superscalar processor model capable of processing multiple instructions per cycle while eliminating false dependencies using register renaming with the help of a Reorde Buffer (ROB).
